\doxylink{structICM20948__IntStatus}{ICM20948\+\_\+\+Int\+Status}


\begin{DoxyItemize}
\item Holds the parsed state of the four interrupt status registers
\item Covers INT\+\_\+\+STATUS (0x19), INT\+\_\+\+STATUS\+\_\+1 (0x1A), INT\+\_\+\+STATUS\+\_\+2 (0x1B), INT\+\_\+\+STATUS\+\_\+3 (0x1C)
\item Populated by check\+Int\+Status() via a single 4-\/byte burst read
\item Reading these registers clears the interrupt flags (when clear\+Mode = 0 in Int\+Pin\+Config)
